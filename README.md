# Salah Discipline

A premium, offline-first mobile platform dedicated to cultivating spiritual discipline through mathematically accurate prayer tracking and reflective practices. Built with a focus on minimalism, the application offers a calm, distraction-free environment that prioritizes deep engagement over cheap gamification.

## Product Vision & Philosophy

Salah Discipline is designed to resemble the thoughtful, clean aesthetics of modern wellness applications (comparable to Apple Health or Calm). It rejects visual clutter, opting instead for a serene interface that encourages mindfulness.

The core philosophy revolves around consistency and reflection. By providing mathematically precise tools and a beautifully structured environment, the app aims to transform daily obligations into mindful, intentional habits. The application operates offline-first, ensuring your spiritual routine remains uninterrupted regardless of connectivity.

## Core Features

- **Mathematically Accurate Prayer Tracking**: Utilizing the robust `adhan` library, the app calculates precise prayer windows based on your coordinates and selected calculation methods.
- **Spiritual Discipline Scoring**: A nuanced metric system tracks your streak, early check-ins, and recovery states without resorting to superficial gamification.
- **Mindful Journaling**: Integrated reflective journaling allows users to document their spiritual journey, tying personal thoughts to their daily practices.
- **Community & Premium Tiers**: Thoughtfully designed community features foster shared discipline, while premium offerings provide deeper analytical insights and advanced tools.

## Technology Stack

Built for scale and performance, the repository follows an enterprise-grade feature-first architecture:

- **Framework**: Expo / React Native
- **Language**: TypeScript
- **State Management**: Zustand (Global State) & TanStack Query (Server State)
- **Styling**: NativeWind (Tailwind CSS) & React Native Reanimated for fluid, native-feeling interactions.
- **Icons**: Lucide React Native
- **Core Logic**: `adhan` (Prayer Times) & `date-fns` (Time Manipulation)
